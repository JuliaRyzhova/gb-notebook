1. Реализация команды DELETE.

Так как эта команда напрямую связана с работой с базой данных, метод delete()
находится в интерфейсе GBRepository.
Этот интерфейс имплементируется классом UserRepository, в котором я реализую
сам метод delete() через @override.
Для того, чтобы удалить строчку из нашего файла db.txt я делаю следующее:
* Создаю список контактов из всех, которые есть в нашей БД, для этого считываю файл.
* Создаю объект "удаляемый контакт" через фильтр по id. Если такой не будет найден,
то вылезет ошибка о том, что контакт отсутствует.
* У списка вызывается метод remove(), в который передается объект удаляемого контакта.
Этот метод удаляет нужный контакт из списка.
* Перезаписываю в файл список с учетом удаленного контакта.

Идентификаторы контакта не равно порядковые номера. Если мы удалим контакт с идентификатором "1",
то идентификаторы остальных контакт не меняются.

Иду дальше в класс UserController и в нем реализую метод deleteUser(), который в свою очередь
вызывает ранее объявленный метод в интерфейсе GBRepository и реализованный в классе
UserRepository.

После этого, уже во view, в классе UserView дописываю case DELETE, где вызывают соответсвующий
метод из класса UserController.
